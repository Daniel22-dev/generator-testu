# Nahrání Generátoru 7.1.0

Rozbalte distribuční ZIP a nahrajte jeho obsah přímo do kořene repozitáře Generátoru. Nechte soubory přepsat, potvrďte commit a vyčkejte na zelené Actions/Pages.

Verze 7.1.0 přidává anonymní technické počítání testových balíčků: úspěch, chyba a zrušení. Obsah testu ani prompt se neukládá.
